# Privacy Policy — FreeIPTV

**Last updated: April 23, 2026**

This privacy policy applies to the FreeIPTV application ("Application") developed by Piotr Mularczyk ("Developer").

---

## Data Collection

FreeIPTV does **not** collect, store, transmit, or share any personal data. The Application does not use analytics, crash reporting services, advertising networks, or any third-party data collection tools.

---

## Permissions

The Application requests the following permissions, used solely for local functionality:

- **INTERNET** — required to download IPTV playlist files (M3U) and EPG data from URLs provided by the user, and to fetch channel lists from the developer's public GitHub repository.
- **WRITE_EXTERNAL_STORAGE** (Android 9 and below) — required to save playlist and template files to the device's Downloads folder.
- **MANAGE_EXTERNAL_STORAGE** (Android 11 and above, optional) — required only when Editor Mode is enabled, to read and write template files in the Downloads folder. This permission is never requested unless the user explicitly activates Editor Mode in the application settings.
- **FOREGROUND_SERVICE / FOREGROUND_SERVICE_MEDIA_PLAYBACK** — required to keep media playback running in the background when the screen is off or the app is minimized.

---

## Local Storage

All data generated by the Application — including playlist files, channel templates, and EPG cache — is stored exclusively on the user's device. No data is uploaded to any server.

---

## Third-Party Services

The Application fetches a publicly available list of IPTV channel files from the developer's GitHub repository (github.com/PiotrMul/freeiptv-templates). No personal data is sent as part of this request.

---

## Children's Privacy

The Application is not directed at children under the age of 13 and does not knowingly collect any information from children.
